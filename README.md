Hangman Game

Overview

This project is a Python-based implementation of the classic Hangman word guessing game. The player attempts to identify a hidden word by guessing one letter at a time within a limited number of attempts.

Features

- Random word selection
- Letter-by-letter word guessing
- Incorrect guess tracking
- Limited number of attempts
- Win and loss conditions
- Simple command-line interface

Technologies Used

- Python 3
- Random Module

Project Structure

- "Hangman-game.py" – Main source code of the application

How to Run

1. Open the project in Pydroid 3 or any Python IDE.
2. Run the "Hangman-game.py" file.
3. Follow the on-screen instructions to play the game.

Learning Objectives

This project demonstrates the use of:

- Functions
- Loops
- Conditional Statements
- Lists and Strings
- User Input Handling
- Basic Game Logic
